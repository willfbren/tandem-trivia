# Tandem Trivia

### Installation

1. Clone the repo
```sh
git clone git@github.com:willfbren/tandem-trivia.git
```
2. Install dependencies
```sh
cd tandem-trivia
npm install
```
3. Start Expo server
```sh
npm start
```
